# User Stories

---

## Onboarding

- As a new user, I want to input my subjects and upcoming exam dates so the app can build me a personalized study plan.
- As a new user, I want to see how the app works differently from flashcards so I understand why these methods are better.
- As a new user, I want to track my energy levels across a week so the app can identify my peak productivity window.

---

## Note Ingestion & Question Generation

- As a user, I want to paste my notes or upload a document so the app can turn them into active recall questions automatically.
- As a user, I want to review and edit AI-generated questions before studying them so I can ensure they match what I need to learn.
- As a user, I want to write my own questions from scratch so I can practice the in-class note-taking technique.
- As a user, I want questions to be categorized by topic and subject so I can stay organized across multiple classes.

---

## Active Recall & Study Sessions

- As a user, I want to start a study session and be shown questions one at a time so I can practice free recall without any cues.
- As a user, I want to type or speak my answer before seeing the correct one so I'm forced to retrieve from memory, not recognize.
- As a user, I want to rate my answer (correct / partial / wrong) so the app knows how well I know each concept.
- As a user, I want a Blurting Mode where I see a topic title and write everything I remember so I can practice free recall on entire subjects.
- As a user, I want the app to compare my blurted answer against the source material and highlight gaps so I know exactly what I missed.

---

## Pre-Testing

- As a user, I want to be quizzed on upcoming material before I've studied it so I can activate the hypercorrection effect.
- As a user, I want to see which pre-test questions I got wrong highlighted during my actual study session so my brain corrects them more deeply.

---

## Feynman Mode

- As a user, I want to type an explanation of a concept in simple terms so I can test whether I truly understand it.
- As a user, I want AI to evaluate my explanation and flag knowledge gaps or jargon I couldn't simplify so I know what to revisit.
- As a user, I want to re-attempt my explanation after reviewing the flagged gaps so I can iterate toward full understanding.

---

## Spaced Repetition

- As a user, I want topics automatically scheduled for review at optimal intervals so I don't have to manage when to revisit material.
- As a user, I want to see a color-coded progress board (green / yellow / red) per topic so I can instantly see what needs work.
- As a user, I want the app to prioritize red topics in my next session automatically so weak areas get more attention.
- As a user, I want a visual forgetting curve per topic so I can see how my retention is projected to decay without review.

---

## Interleaved Practice

- As a user, I want my study sessions to automatically mix questions from different subjects so I get the benefits of interleaved practice.
- As a user, I want the app to prompt me to switch topics after my chosen interval (25–90 min) so I don't stay on one subject too long.

---

## Study Session Timer & Breaks

- As a user, I want a Pomodoro-style timer during study sessions so I work in focused intervals.
- As a user, I want random 10-second gap effect prompts during sessions so my brain can replay what I just learned.
- As a user, I want to be reminded to take a break after each session so I can consolidate learning through rest.

---

## Study Plan & Scheduling

- As a user, I want AI to build a prospective revision timetable based on my subjects, weak areas, and exam dates so I always know what to study.
- As a user, I want the plan to update dynamically based on my performance so weaker topics get more time automatically.
- As a user, I want to see a weekly calendar view of my scheduled study sessions so I can plan my time around them.

---

## Progress & Analytics

- As a user, I want a dashboard showing my retention score per topic over time so I can track improvement.
- As a user, I want to see how many sessions I've completed and how long I've studied so I can monitor my consistency.
- As a user, I want to see which topics have been neglected the longest so I know what's at risk of being forgotten.
