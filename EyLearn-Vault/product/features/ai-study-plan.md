# AI Study Plan Builder

**Type:** Content & Setup

## What it does
AI generates a prospective revision timetable based on subjects, exam dates, current topic health, and the user's peak productivity window.

## Inputs
- Subjects and topics
- Exam dates
- Weak areas (red/yellow topics from [[spaced-repetition]])
- Peak productivity window (from [[peak-window-tracker]])

## Outputs
- Weekly calendar with scheduled sessions
- Each session specifies: subjects, estimated duration, focus areas
- Plan updates dynamically as performance data changes

## Related
- [[spaced-repetition]] — overdue and weak topics prioritized in plan
- [[peak-window-tracker]] — hardest sessions scheduled during peak window
- [[progress-dashboard]] — plan visible from dashboard
- [[forgetting-curve]] — topics near forgetting threshold scheduled first
