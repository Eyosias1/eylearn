# Future Features Index

Modes and features not yet built. Ranked roughly by value vs. effort.

## Study Session Modes

| Mode | Value | Effort | Notes |
|---|---|---|---|
| [[cloze-deletion]] | High | Low | Fill-in-the-blank from your notes. Good warmup, easy to build |
| [[case-based]] | High | High | AI generates scenarios from any notes. Most differentiated |
| [[socratic-dialogue]] | Medium | Medium | Multi-turn AI dialogue. Overlaps with Feynman |
| [[mind-map-reconstruction]] | Low | High | Rebuild concept map from memory. Research favors retrieval over mapping |

## Build Order Recommendation
1. **Cloze deletion** — highest impact, lowest effort, fills gap for vocab-heavy subjects
2. **Case-based** — most unique, hardest to build, highest long-term value
3. **Socratic dialogue** — only if Feynman proves insufficient for reasoning depth
4. **Mind map** — last, only if there's demand and a node editor is already in the app
