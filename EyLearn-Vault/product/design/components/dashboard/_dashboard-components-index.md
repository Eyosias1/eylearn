# Dashboard Components Index

All components for the `/` dashboard page. Plan each file before building.
Wireframe: [[dashboard]]

- `DailyTargets` — recall tasks done/total + SR due count + progress bar
- `ConsistencyChart` — 14-day bar chart of daily study activity
- `NextSessionCard` — AI-recommended next session + launch CTA (most prominent element)
- `TodaySequence` — time-blocked list of today's planned sessions
- `ActiveSubjectsGrid` — 2-col grid of subjects with mastery % + health color
- `GlobalStatsBar` — total focus time, streak, items memorized
