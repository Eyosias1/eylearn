# Study Plan Components Index

All components for the `/study-plan` page. Plan each file before building.
Wireframe: [[study-plan]]

- `WeeklyCalendar` — 7-day grid with time-blocked session blocks, click to create/edit
- `MonthlyCalendar` — condensed month view with dot indicators, toggled from weekly
- `SpacedRepSidebar` — retrieval strength bar + upcoming due queue with RECALL NOW / SCHEDULE actions
- `InterleavingTools` — subject checkboxes with cognitive load badges, reveals interleaved session CTA at 2+ selections
- `SessionIntentions` — current focus display + next intention input, shown at session start
