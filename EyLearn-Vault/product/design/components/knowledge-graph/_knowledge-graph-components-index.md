# Knowledge Graph Components Index

All components for the `/graph` page. Plan each file before building.
Wireframe: [[knowledge-graph]]

- `GraphCanvas` — force-directed node graph, pan/zoom, hover + click interactions (D3/react-force-graph)
- `GraphToolbar` — subject filter, mastery filter, labels toggle
- `NodeSidePanel` — topic detail panel (mastery, last reviewed, connections, Start Session CTA)
- `GraphLegend` — color key overlaid on canvas bottom-left
